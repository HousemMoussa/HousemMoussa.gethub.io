ASTOVL Aerodynamics Flight Model

Project Overview

This project focused on developing a comprehensive aerodynamics model and simulation framework for an Advanced Short Take-Off and Vertical Landing (ASTOVL) lift-fan aircraft. The model encompasses power-off aerodynamic forces and moments, propulsion system-induced aerodynamic effects, and unpowered in-ground effects, specifically targeting low-speed flight regimes (Mach number M ≈ 0.2).

Aircraft Description

The aircraft is a single-place, single-engine fighter/attack aircraft with a wing-canard arrangement and twin vertical tails. It features a remote lift fan interconnected with a lift cruise turbofan engine, allowing for vertical take-off and landing capabilities.

Key Equations

Longitudinal Aerodynamics

Lift:

L = CLq̄S + ΔLTT

Drag:

D = CDq̄S

Pitching moment:

PM = Cmq̄Sc̄ + ΔPMTdeTde + (Lcosα + Dsinα)XMRC + (Lsinα - Dcosα)ZMRC

Lateral Directional Aerodynamics

Side force:

FY = Cyq̄S

Rolling Moment:

RM = Clq̄Sb + ΔRMTdeTde + FYZMRC

Yawing Moment:

YM = Cnq̄Sb + FYXMRC

Implementation

The aerodynamics model was implemented using MATLAB and Simulink. The MATLAB code contains all the necessary data and equations, while Simulink provides a block diagram environment for multidomain simulation and Model-Based Design.

Key Findings

  • The angle of attack significantly influences the pitching moment, with oscillations in AoA creating corresponding oscillations in the pitching moment.
  • The sideslip angle has a substantial impact on side force, yawing moment, and rolling moment.
  • Increasing speed automatically intensifies drag, as expected.
  • In hover-to-forward flight transitions, scalar variations in forces were observed due to changes in speed and nozzle angle.

Conclusion

The development of this aerodynamic model using MATLAB & Simulink allowed for comprehensive testing of the ASTOVL aircraft. The model provides valuable insights into the aircraft's behavior during transition and hover flight phases, focusing on low-speed aerodynamics. This project serves as a foundation for future investigations into ASTOVL situations and can be further enhanced by introducing wind forces and integrating with advanced flight simulators.
